# Laboratorio 15 - Implementación de Pasarela de Pago

## Descripción

En este laboratorio se continuó con el desarrollo de la aplicación de e-commerce, implementando nuevas funcionalidades relacionadas con el consumo de datos, la administración del estado global y el flujo de compra de películas. Se utilizó la API de TMDB para obtener la información de las películas, se incorporó Zustand para la gestión del estado global de la aplicación, se desarrollaron todas las páginas del sistema, se implementó una simulación de pasarela de pagos para la compra de películas y se agregaron pruebas para validar el correcto funcionamiento del proyecto.
